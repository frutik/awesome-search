# Algolia

Search-as-a-service company providing hosted search infrastructure for e-commerce, SaaS, and content platforms. One of the largest commercial search providers.

## Notable Contributions

**A/B Testing for Search** — Algolia has written extensively about metric selection for search A/B tests, introducing the **directionality vs. sensitivity** framework (from Yandex Research) to help practitioners select appropriate success metrics.

**Key insight**: Directional metrics (revenue) are slow to reach significance; sensitive metrics (CTR) are fast but gameable. The right approach combines both levels.

**NeuralSearch / Hybrid Search** — Algolia provides combined keyword + semantic search as a hosted product, positioning toward AI-native search experiences.
