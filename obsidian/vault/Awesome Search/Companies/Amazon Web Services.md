# Amazon Web Services

Amazon's cloud platform. In the search/IR space AWS operates **Amazon OpenSearch Service** — the managed offering built on [[OpenSearch]], the Apache-2.0 fork AWS created from [[Elasticsearch]] 7.10 in 2021.

## Search Products & Services

- **[[OpenSearch]]** — the open-source engine AWS forked and stewards.
- **Amazon OpenSearch Service** — managed OpenSearch (Serverless and Managed Clusters) with IAM/VPC/CloudWatch integration.
- **Migration Assistant** — tooling to move self-managed [[Solr]] / Elasticsearch / OpenSearch clusters to Amazon OpenSearch Service; now offers an AI-assisted, agent-guided experience.
- **Amazon Bedrock** — managed foundation-model service used for embedding inference and RAG alongside OpenSearch.
